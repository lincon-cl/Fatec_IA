## 🧠 Visão Geral do Repositório

Este ecossistema está dividido em três grandes pilares:
1. **Data Science & Analytics:** Limpeza de dados, feature engineering e análise exploratória (EDA).
2. **Machine Learning Clássico:** Algoritmos de regressão, classificação e agrupamento (Scikit-Learn).
3. **Deep Learning & GenAI:** Redes neurais (CNNs, RNNs, Transformers) e integração com LLMs (OpenAI, Hugging Face).
